J'apprends à Scraper !

Le HTML est le langage de balisage utilisé pour structurer les données sur les pages web. Chaque élément d’une page est défini par une balise HTML. En tant que scraper, vous devez savoir comment identifier les informations pertinentes dans cette structure.

Les éléments HTML les plus courants :

<div>

Une balise de division, souvent utilisée comme conteneur pour d’autres éléments. Elle ne représente pas d'information en elle-même mais est utile pour organiser le contenu.

<span>

Une balise en ligne utilisée pour cibler de petites portions de texte ou d’autres éléments. Souvent utilisée en conjonction avec des classes CSS pour le style.

Exemple d'utilisation : Texte stylé avec un span.

<a>

La balise d'ancrage, utilisée pour les liens hypertexte. C'est l'une des balises les plus importantes à comprendre, car elle contient souvent des URL et des liens internes/externes.

Exemple de lien : Visitez Example.com

<img>

La balise d'image, utilisée pour afficher des images. Elle contient un attribut src avec l’URL de l’image.
