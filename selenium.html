Exercice 1 : Ouvrir une page web et récupérer son titre

Objectif : Ouvrir une page web avec Selenium et afficher son titre dans la console.

from selenium import webdriver

driver = webdriver.Chrome()
driver.get("https://formation-scraping.netlify.app/")
print(driver.title)
driver.quit()

Exercice 2 : Cliquer sur un lien

Objectif : Localiser et cliquer sur un lien de la page en utilisant By.ID ou By.CLASS_NAME.

from selenium import webdriver
from selenium.webdriver.common.by import By
import time

driver = webdriver.Chrome()
driver.get("https://formation-scraping.netlify.app/")
time.sleep(3)

try:
    link = driver.find_element(By.CLASS_NAME, "selenium-link")
    link.click()
    print("Lien cliqué.")
except Exception as e:
    print(f"Erreur : {e}")

driver.quit()

Exercice 3 : Remplir un formulaire

Objectif : Localiser un champ de formulaire par ID et y entrer du texte.

from selenium import webdriver
from selenium.webdriver.common.by import By

driver = webdriver.Chrome()
driver.get("https://formation-scraping.netlify.app/form_page.html")

username_field = driver.find_element(By.ID, "username")
username_field.send_keys("mon_nom_utilisateur")

password_field = driver.find_element(By.ID, "password")
password_field.send_keys("mon_mot_de_passe")

driver.quit()
